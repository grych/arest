    new(base_url, **options) click to toggle source

    Input:

    • base_url - the base URL for REST API

    Options:

    • headers (Hash) to set up headers c = ::new “localhost:3000/_api/v1”, headers: {“Authentication” => “Token token=AHDIUHSBC.…”}

    • username and password for basic http authentication c = ::new “localhost:3000/_api/v1”, username: “user1”, password: “password.8”

    • Authentication token c = ::new “localhost:3000/_api/v1”, token: 'AdbsuUDhwb3hqhj…'

def initialize(base_url, **options)
  @base_url = base_url
  @headers = options[:headers]
  @auth_user = options[:username]
  @auth_password = options[:password]
  @auth_token = options[:token]
end
